# Apex_rl

A reinforcement learning library.

Documentation: https://apex-rl-doc.readthedocs.io/

## Installation

Clone and install from source:

```bash
git clone https://github.com/Atticlmr/Apex_rl.git
cd Apex_rl
pip install -e .
```

or use uv

```bash
git clone https://github.com/Atticlmr/Apex_rl.git
cd Apex_rl
uv pip install -e .
```
